This meeting will touch on the following points:

1) What is TraDIS(-XPress)?
2) QuaTraDIS: our in-house TraDIS analysis pipeline \
	2.1) From sequences to plot file\
	2.2) From plot files to gene lists\
	2.3) New features\
		2.3.1) Dynamic window approach\
		2.3.2) New characterisation approach
3) Downstream analysis\
	3.1) User interface\
	3.2) Pathway analysis\
	3.3) Synergy

The presentation will be accompanied by a demo of the main pipelines. Data used in the demo as well as installation and user instructions can be found on the hpc in sib-hpc-data/informatics/transfer/incoming/sbastkowski/DataScienceTraining2025.
